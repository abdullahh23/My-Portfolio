# Abdullah Bin Yousaf — Premium Portfolio

A premium, custom-designed dark theme portfolio built with semantic HTML, modern CSS variables, and GSAP animations. Fully responsive and optimized for mobile devices.

## 🚀 Live Preview & Deployment

This project is a static web application and can be deployed to production instantly.

### Option 1: Deploy to Vercel (Recommended - Free & Instant)

Vercel is the easiest way to deploy static websites. It's completely free and takes less than a minute.

1. Go to [vercel.com](https://vercel.com) and log in with your GitHub account.
2. Click **Add New** and select **Project**.
3. Import your repository: `My-Portfolio`.
4. Click **Deploy**. Vercel will automatically build the site and provide a shareable production URL (e.g., `abdullah-portfolio.vercel.app`).

### Option 2: Deploy to GitHub Pages (Free)

Since this repository is already on GitHub, you can publish it directly:

1. Open your repository on GitHub.
2. Go to **Settings** -> **Pages**.
3. Under **Build and deployment**, select **Deploy from a branch**.
4. Choose `master` branch and `/ (root)` folder.
5. Click **Save**. Within 1–2 minutes, your site will be live at `https://abdullahh23.github.io/My-Portfolio/`.

## 🛠️ Local Development

To run the project locally:

1. Clone this repository to your local machine.
2. Run a simple static HTTP server:
   ```bash
   # Using Python
   python -m http.server 8080
   
   # Using Node.js (any static server)
   npx serve .
   ```
3. Open `http://localhost:8080` in your web browser.
